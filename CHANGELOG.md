## [0.1.0] - 2025-02-08

### Added
- Initial release of Bazel MCP Server
- Support for `bazel_build` tool - Build targets with options
- Support for `bazel_test` tool - Run tests with filtering
- Support for `bazel_clean` tool - Clean build outputs
- Support for `bazel_run` tool - Build and run targets
- Support for `bazel_query` tool - Query dependency graph
- Support for `bazel_aquery` tool - Query action graph
- Support for `bazel_info` tool - Display Bazel runtime info
- Comprehensive test coverage
- CI/CD with GitHub Actions
- Cross-platform release binaries (Linux, macOS, Windows)
- Complete documentation and examples
